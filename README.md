# 💻 CPU Scheduling Simulator

An interactive web-based simulator to visualize and analyze CPU Scheduling Algorithms.

## 🚀 Features

* Supports multiple scheduling algorithms:

  * First Come First Serve (FCFS)
  * Shortest Job First (SJF)
  * Shortest Remaining Time First (SRTF)
  * Round Robin (RR)
* Dynamic Gantt Chart Visualization
* Performance Metrics:

  * Turnaround Time
  * Waiting Time
* Comparison Mode (All algorithms side-by-side)
* Adjustable Time Quantum (for Round Robin)
* Add / Remove / Random Process Generator
* Clean and modern UI

---

## 🧠 Algorithms Implemented

* FCFS (Non-preemptive)
* SJF (Non-preemptive)
* SRTF (Preemptive)
* Round Robin (Preemptive)
